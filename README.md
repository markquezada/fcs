# FCS: FreeSWITCH Command Socket

FCS is a lightweight api wrapper for [FreeSWITCH's event socket][event_socket].

## Installation

Add this line to your application's Gemfile:

    gem 'fcs'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fcs
